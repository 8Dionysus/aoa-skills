# Status promotion reviews

This folder stores public review records for non-canonical status promotions.

Use this surface for promotion work such as:
- `scaffold -> evaluated`
- `linked -> reviewed`
- `reviewed -> evaluated`

Do not use this folder for `canonical` candidate work.
That remains in `../canonical-candidates/`.

New reviews in this folder should use:

`templates/STATUS_PROMOTION_REVIEW.template.md`
