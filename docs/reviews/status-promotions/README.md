# Status promotion reviews

This folder stores public review records for non-canonical status promotions.

Use this surface for promotion work such as:
- `scaffold -> evaluated`
- `linked -> reviewed`
- `reviewed -> evaluated`

Do not use this folder for `canonical` candidate work.
That remains in `../canonical-candidates/`.

New reviews in this folder should use:

`templates/STATUS_PROMOTION_REVIEW.template.md`

Keep the review record truth-sync friendly.
At minimum it should state the current status, governance decision, scope, lineage,
reviewed revision, recorded governance outcome, whether runtime `SKILL.md` meaning changed,
and blockers for both this target status and the next status step.
